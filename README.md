# Практична робота №9 — Mini REST API in Node.js

## Мета роботи

Навчитися створювати простий REST API на чистому Node.js без використання сторонніх фреймворків, працювати з HTTP-запитами, JSON-даними та файловою системою.

## Виконані завдання

У ході виконання практичної роботи було реалізовано:

- отримання списку ресурсів (REST LIST);
- отримання ресурсу за id (REST GET BY ID);
- створення нового ресурсу (REST CREATE);
- оновлення існуючого ресурсу (REST UPDATE);
- видалення ресурсу (REST DELETE).

## Як запускати рішення

bash
node rest_list.js 3000
node rest_get_by_id.js 3000
node rest_create.js 3000
node rest_update.js 3000
node rest_delete.js 3000
```

## Структура проєкту
rest_list.js
rest_get_by_id.js
rest_create.js
rest_update.js

data.json
README.md
```
